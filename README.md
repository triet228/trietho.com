# Contribute to [Triet cat gallery](https://www.trietho.com/fun_fact/)

Please make a pull request with the following change.
* Add new image(s) of Triet cat in `cat<number>.jpg` format at `trietho.com/static/trietcat/`
* At the end of file `trietho.com/content/fun_fact.md`, add this line:
```
<img src="/trietcat/cat<number>.jpg" alt="Triet's Cat" width="500" style="background: white; padding: 0px; border-radius: 10px; margin: 20px auto; display: block;">
```
Note: Remember to replace `<number>` with the actual image number.
